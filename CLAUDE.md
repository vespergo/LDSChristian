# CLAUDE.md

This file provides guidance to Claude Code (claude.ai/code) when working with code in this repository.

## Project Overview

A multi-page website comparing LDS (Latter-day Saints) theology with mainstream Christian traditions (Catholic, Protestant, Orthodox). Built with **Eleventy (11ty)** using Nunjucks (`.njk`) templates.

## Architecture

- **Source files:** `src/*.njk` — one file per page, each extending `src/_includes/base.njk`
- **Base template:** `src/_includes/base.njk` — shared nav, `<head>`, and JS toggle; renders `{{ title }}`, `{% block pageStyles %}`, `{% block content %}`, `{% block pageScript %}`
- **Build output:** `_site/` (generated by Eleventy)
- **Styles:** Each page defines its own `<style>` block inside `{% block pageStyles %}` — no shared stylesheet

**Frontmatter fields per page:**
- `title` — `<title>` and page heading
- `navPage` — slug matched in base.njk to highlight the active nav link (e.g. `practices`)
- `navGroup` — highlights the active dropdown group: `comparisons`, `lds-faith`, `reference`, `stories`

**Pages (`src/`):**
- `index.njk` — home/overview with similarity/difference card grid
- `are-lds-christian.njk` — focused essay on that question
- `catholic.njk`, `protestant.njk`, `orthodox.njk` — comparison pages per tradition
- `closest.njk` — which tradition is theologically closest to LDS
- `conference.njk` — General Conference content
- `practices.njk` — LDS religious practices
- `history.njk` — LDS church history
- `glossary.njk` — terminology definitions
- `faq.njk` — frequently asked questions
- `mission.njk` — serving a mission
- `temple.njk` — LDS temples
- `patriarchal-blessings.njk` — patriarchal blessings
- `scriptures.njk` — key scriptures
- `first-vision.njk` — Joseph's First Vision
- `old-testament-temples.njk` — OT temple context
- `ark-of-the-covenant.njk` — Ark of the Covenant story
- `testimony.njk` — How to gain a testimony
- `joseph-smith.njk` — The Prophet Joseph Smith
- `word-of-wisdom.njk` — The Word of Wisdom health code

**Shared design system (defined per-page in `{% block pageStyles %}`):**
- CSS custom properties in `:root`: `--blue`, `--gold`, `--green`, `--red`, `--bg`, `--card`, `--text`, `--muted`, `--radius`, `--shadow`
- Top nav: `.topnav`, `.nav-group`, `.nav-toggle`, `.nav-dropdown` dropdown pattern
- Card components: `.card.sim` (green border) for similarities, `.card.diff` (red border) for differences
- Section labels: `.section-label.sim` / `.section-label.diff`

## Development

```powershell
# Install dependencies (first time)
npm install

# Build once
npx eleventy

# Build + watch
npx eleventy --serve
```

## Key Conventions

- To add a new page: create `src/<slug>.njk`, then add its `<a>` link to the correct dropdown in `src/_includes/base.njk`.
- Nav changes only need to be made in **one file**: `src/_includes/base.njk`.
- Copy the full `{% block pageStyles %}` CSS from an existing page (e.g. `practices.njk`) as a starting point for new pages.
- The site has a permission allow-list for `WebFetch(domain:www.churchofjesuschrist.org)` in `.claude/settings.local.json`.
